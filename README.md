简介
------
* 支持命令同步处理
* 支持命令同步转异步处理
* 支持命令调度处理
* 支持命令运行时监控
* 支持命令失败重试

适用场景
------
* 分布式系统下因网络原因导致的重试场景
* 业务链路实时性要求不高场景

接入方式
------
* maven工程

组件设计
------

* 线程池

* 命令处理器注册初始化

* 监控

* 调度

------
组件的运行需要结合于Mybatis ORM框架，其中同步任务模型持久化目前考虑使用数据库作为存储介质。
